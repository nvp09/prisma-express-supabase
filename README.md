# Prisma + Next.js + Supabase

This project demonstrates how to use Prisma ORM with Next.js and Supabase PostgreSQL.

## Tech Stack

- Next.js
- Prisma ORM
- Supabase PostgreSQL
- TypeScript

## Setup

Install dependencies:

npm install

Setup environment variables:

cp .env.example .env

Run development server:

npm run dev

## API Endpoints

GET /api/categories  
GET /api/posts
